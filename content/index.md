Potatso 是一款网络代理工具，目前仅适用于 iOS 平台，支持以下功能：

{{< note title="功能" >}}
1. 自定义代理：支持主流的代理类型，如 Socks 5，Shadowsocks 
2. 自定义规则，自由管理网络请求
3. 自定义 DNS
4. 多配置切换
5. 网络请求记录
6. iCloud 同步
{{< /note >}}
